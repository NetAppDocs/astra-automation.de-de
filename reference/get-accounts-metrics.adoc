= Alle Kennzahlen für ein Konto abrufen
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{account_id}/metrics`#

Gibt alle Kennzahlen für das Konto zurück



== Parameter

[cols="5*"]
|===
| Name | Typ | In | Erforderlich | Beschreibung 


| Konto-ID | Zeichenkette | Pfad | Wahr  a| 
ID der übergeordneten Kontoressource

* Beispiel: {{.Account}}


|===


== Antwort

[source]
----
Status: 200, Returns a OpenMetrics text document containing all metrics for the account.
----
