= Konto löschen
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#LÖSCHEN# [.api-doc-code-block]#`/accounts/{account_id}`#

Entfernt die angegebene Account-API-Ressource.

*Nebenwirkungen*

* "deletePending" ist auf "true" gesetzt und auf das Konto kann nicht mehr zugegriffen werden.




== Parameter

[cols="5*"]
|===
| Name | Typ | In | Erforderlich | Beschreibung 


| Konto-ID | Zeichenkette | Pfad | Wahr  a| 
ID der aufzulistenden Kontensammlung

* Beispiel: {{.Account}}


|===


== Antwort

[source]
----
Status: 204, Indicates if the account resource was deleted.
