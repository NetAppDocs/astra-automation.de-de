= Astra API (ACC) Anwendungs-Asset-Endpunkte
:allow-uri-read: 


Die Application Asset API ermöglicht den Zugriff auf interne Sammlungen von Statusinformationen, die zur Verwaltung von Anwendungen verwendet werden.
