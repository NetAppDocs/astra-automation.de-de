= Löschen eines Anwendungssnapshots
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#LÖSCHEN# [.api-doc-code-block]#`/accounts/{account_id}/k8s/v1/apps/{app_id}/appSnaps/{appSnap_id}`#

Entfernt die angegebene Application Snapshot API-Ressource.

*Nebenwirkungen*

* Falls gerade ein Snapshot erstellt wird, wird dieser abgebrochen.
* Wenn ein Snapshot vollständig ist, werden die Snapshot-Daten gelöscht.




== Parameter

[cols="5*"]
|===
| Name | Typ | In | Erforderlich | Beschreibung 


| Konto-ID | Zeichenkette | Pfad | Wahr  a| 
ID der übergeordneten Kontoressource

* Beispiel: {{.Account}}




| App-ID | Zeichenkette | Pfad | Wahr  a| 
ID der übergeordneten App-Ressource



| appSnap_id | Zeichenkette | Pfad | Wahr  a| 
ID der appSnap-Sammlung, die aufgelistet werden soll

|===


== Antwort

[source]
----
Status: 204, Indicates if the appSnap resource was deleted.
