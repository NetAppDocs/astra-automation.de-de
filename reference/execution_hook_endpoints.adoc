= Astra API (ACC) Ausführungs-Hook-Endpunkte
:allow-uri-read: 


Die Execution Hook API ermöglicht die einfache Verwaltung von Ausführungshooks. Ausführungshooks enthalten zusätzliche Konfiguration und Verweise auf eine Hook-Quellressource.
