= Eine Gruppe für einen Benutzer löschen
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#LÖSCHEN# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/users/{user_id}/groups/{group_id}`#

Entfernt die angegebene Group-API-Ressource.

*Nebenwirkungen*

* Die folgenden Ressourcen, die der Gruppe zugeordnet sind, werden gelöscht:
* Rollenbindungen, die sich auf die Gruppe beziehen




== Parameter

[cols="5*"]
|===
| Name | Typ | In | Erforderlich | Beschreibung 


| Konto-ID | Zeichenkette | Pfad | Wahr  a| 
ID der übergeordneten Kontoressource

* Beispiel: {{.Account}}




| Benutzer-ID | Zeichenkette | Pfad | Wahr  a| 
ID der übergeordneten Benutzerressource



| Gruppen-ID | Zeichenkette | Pfad | Wahr  a| 
ID der aufzulistenden Gruppensammlung

|===


== Antwort

[source]
----
Status: 204, Indicates if the group resource was deleted.
