= Löschen eines Zertifikats
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#LÖSCHEN# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/certificates/{certificate_id}`#

Entfernt die angegebene Certificate API-Ressource.



== Parameter

[cols="5*"]
|===
| Name | Typ | In | Erforderlich | Beschreibung 


| Konto-ID | Zeichenkette | Pfad | Wahr  a| 
ID der übergeordneten Kontoressource

* Beispiel: {{.Account}}




| Zertifikats-ID | Zeichenkette | Pfad | Wahr  a| 
ID der aufzulistenden Zertifikatssammlung

|===


== Antwort

[source]
----
Status: 204, Indicates if the certificate resource was deleted.
----
