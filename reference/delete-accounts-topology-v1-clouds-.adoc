= Eine Cloud löschen
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#LÖSCHEN# [.api-doc-code-block]#`/accounts/{account_id}/topology/v1/clouds/{cloud_id}`#

Löscht die angeforderte Cloud-Instanz sowie alle nicht verwalteten Cluster und Volumes, die mit dieser spezifischen Cloud verknüpft sind.



== Parameter

[cols="5*"]
|===
| Name | Typ | In | Erforderlich | Beschreibung 


| Konto-ID | Zeichenkette | Pfad | Wahr  a| 
ID der übergeordneten Kontoressource

* Beispiel: {{.Account}}




| cloud_id | Zeichenkette | Pfad | Wahr  a| 
ID der aufzulistenden Cloud-Sammlung

|===


== Antwort

[source]
----
Status: 204, Indicates if the cloud resource was deleted.
