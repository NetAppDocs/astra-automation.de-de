= Astra API (ACC) Zertifikatsendpunkte
:allow-uri-read: 


Die Certificate API repräsentiert vom Administrator als vertrauenswürdig eingestufte Zertifikate, die zum Aufbau ausgehender Verbindungen verwendet werden.
