= Astra API (ACC) Namespace-Endpunkte
:allow-uri-read: 


Die Namespace-API bietet schreibgeschützten Zugriff auf Namespace-Ressourcen, um Namespaces zu beschreiben, die einem bestimmten Cluster zugeordnet sind.
